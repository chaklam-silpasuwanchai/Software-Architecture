# How to get started

- VuePress Guide https://vuepress.vuejs.org/

## Run local server
- PS. install Docker first

```sh
docker-compose up --build -d
```
```sh
docker-compose exec project bash
```
```sh
yarn dev
```

Happy coding.

## Stop local server
```sh
docker-compose down
```

## Clean Docker 
```sh
docker system prune
```
